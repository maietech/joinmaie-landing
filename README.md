# Media, AI Exchange (MAIE) Landing Page

## 📖 Overview
Welcome to the landing page repository for MAIE (Media, AI Exchange). This repository houses the frontend static assets (`index.html`, `styles.css`, `theme.js`, and `reveal.js`) that power our public-facing splash page at [joinmaie.com](https://joinmaie.com).

The landing page serves as the primary public-facing introduction to MAIE and provides visitors with an overview of the platform, its vision, and opportunities to learn more or join the MAIE community.
